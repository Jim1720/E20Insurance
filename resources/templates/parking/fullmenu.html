Register

New Customers must first register to use this site. You will have to key a customer id and a password. Please make a note of them.
You then must select a plan before you can enter claims. This will allow a portion of the service cost to be covered for the new claims.

Register

Sign-In

Existing Customers must sign in to use this site. You only have to register once. Once signed in you can submit claims; review claim history; pay and adjust medical, dental, vision and drug claims.

Sign in

Update

Existing Customers may update their information.

Claim

You can enter new claims. Use numerics for diagnosis and procedures. Four claim types are allowed: Medical, Dental, Vision and Drug.
Claims can later be adjusted from the claim history screen.

History

Claim History screen allows review adjustment and payment of claims.

Style

The Style link changes screen styles.
Click to add a picture, outline or solid background for a specific screen. When solid or outline is used, click the color link to change colros for that screen. All styles can be reset by using -> info -> reset style button.

Administrator

The administrator can reset customer passwords, change the customer identification and list all customers in the database.
